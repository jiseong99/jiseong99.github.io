Phishing Website Detection (ML/DL Framework)

ASU CSE 572 Data Mining Project
GitHub
Stacking Ensemble Recall 0.967 · xDeepFM Recall 0.993

Built a real-time phishing detection system using 112 URL, host, and TLS certificate features across 88,647 samples. Compared tree-based models (Random Forest, XGBoost, LightGBM, CatBoost) against deep learning models (MLP, TabNet, DeepFM, xDeepFM), and optimized Soft Voting and Stacking ensembles with Recall as the primary metric to minimize false negatives. Text-to-SVG Generation with Fine-Tuned LLMs

ASU CSE 575 Final Project
GitHub
Mistral 7B: CLIP Score 0.274, ~9 sec per prompt (orders of magnitude faster than diffusion methods)

Fine-tuned open-source LLMs — LLaMA (8B/12B), Mistral 7B, Gemma, and Qwen — with LoRA/QLoRA to generate SVG markup directly from natural language descriptions. Trained on the 2.17M-pair text2svg-stack dataset, using 4-bit quantization and gradient checkpointing to make training feasible on limited academic hardware. Fine-tuned Mistral 7B produced syntactically valid, semantically aligned SVGs competitive with diffusion-based methods (SVGDreamer, DiffSketch) while generating results dramatically faster.

HelloPick! VOD Recommendation Algorithm

LG HelloVision Final Project (Team)
GitHub
Designed a custom Extended SVDpp model · Precision@10 of 0.077

The dataset had no explicit rating column, so an initial watch-rate-based proxy was used — but it produced weak Precision@k. Diagnosed the root cause and designed an Extended SVDpp model that folds six engineered signals (recency weighting, series completion rate, content popularity, and more) directly into the model's hyperparameters, optimized via Optuna. Built the full pipeline from TMDB poster crawling to collaborative filtering model comparison (ALS, SVD, SVDpp, etc.) using the Surprise library.

Smart Quality Management for Injection Molding

Manufacturing Data Hackathon (Team IESC)
GitHub
Yield improved 73% → 79%, error rate reduced 16% → 1.5%

Worked with 26.8M rows of injection molding process data that had no quality labels. Generated Normal/Abnormal pseudo labels using control-chart-driven analysis, then trained a Recall-optimized predictive model with PyCaret AutoML under extreme class imbalance (1M normal vs. 28K abnormal). Combined this with ANSI/ASQ Z1.4 statistical sampling inspection and validated the pipeline against real manufacturing KPIs — yield and cost — rather than offline accuracy alone.

LG Aimers Phase 2 (Ranked 63rd of 844 teams)
